4. Data Storage and Security

Security Measures: We implement industry-standard security practices to protect your data, including encryption, secure authentication, and regular security audits.

4.1 Local Storage

Some data is stored locally on your device using secure storage mechanisms. This includes cached content, preferences, and temporary data.

4.2 Server Storage

Account data, conversations, bot information, and user-generated content are stored on secure servers with appropriate access controls and encryption.

4.3 Encryption

User-to-user messages may be encrypted using AES-256-GCM encryption. We use secure key derivation (PBKDF2) and implement end-to-end encryption where technically feasible.
